# 📝 NoteHub

A simple and clean **Flutter Note App** built to practice CRUD (Create, Read, Update, Delete) operations with a well-organized project structure.

## ✨ Features

* 🔐 Login & Register
* 📝 Add Notes
* 📖 View Notes
* ✏️ Edit Notes
* 🗑️ Delete Notes
* 👤 Profile Screen
* 🚀 Splash Screen
* 🎨 Clean & Responsive UI

## 🛠️ Built With

* Flutter
* Dart
* Material Design
* Custom Widgets

## 📂 Project Structure

* **screens/** – App screens
* **widgets/** – Reusable UI components
* **models/** – Data models
* **services/** – Authentication & database logic
* **data/** – Local note data
